1. 题目背景
《与星共舞》（Dancing with the Stars，简称 DWTS）是美国版的国际电视选秀节目，源自英国节目《舞动奇迹》（Strictly Come Dancing）。该节目的版本已在阿尔巴尼亚、阿根廷、澳大利亚、中国、法国、印度等许多国家播出。本题关注的是美国版，该版本已完成 34 季。
在节目中，名人与专业舞者搭档，每周进行舞蹈表演。专家评委小组为每对选手的舞蹈打分，观众则通过电话或网络为他们喜爱的搭档投票。观众每周可以投票一次或多次，具体次数以每周公布的限制为准。此外，观众投票支持他们希望留下的明星，但不能投票淘汰某位明星。评委分数和观众投票相结合，以决定当周淘汰哪对选手（综合得分最低者）。三对（某些赛季为更多）选手进入决赛，在决赛周，观众和评委的综合得分用于确定他们从第 1 名到第 3 名（或第 4 名）的最终排名。
结合观众投票和评委分数的方法有很多种。在美国节目前两季中，结合方式基于排名。第二季出现的争议（由于名人参赛者杰里·赖斯尽管评委分数很低却成为决赛选手）导致节目修改为使用百分比而非排名。附件中提供了这两种方法的示例。
在第 27 季，当名人选手鲍比·博恩斯尽管评委分数持续偏低却赢得冠军时，再次引发了"争议"。作为回应，从第 28 季开始对淘汰流程进行了轻微修改。综合评委分数和观众投票确定排名最后的两位选手，然后在直播节目中由评委投票选择淘汰这两位中的哪一对。大约在同一时期，制片人还恢复了使用前两年的排名法来结合评委分数和观众投票。恢复使用排名法的确切赛季尚不确定，但合理假设是在第 28 季。
- 评委分数旨在反映哪位舞者的技术更好，尽管对于什么使舞蹈更好存在一定的主观性。
- 观众投票可能更具主观性，受舞蹈质量影响，但也受名人的人气和魅力影响。
- 节目制片人实际上可能更喜欢评委和观众意见及投票之间的冲突，因为此类事件会提升观众兴趣和收视率。

  - 您产生的观众投票总数的确定性如何，每位参赛者/每周的确定性是否总是相同？提供估算的确定性衡量标准。
1. 总目标：对每一赛季每一周建立模型，估算：$$v_{i,t}$$，即每周每位选手真实观众投票数
2. 数据预处理：评委总分
  评委人数为3-4，为了消除评委人数、打分尺度差异，需要进行归一化：$$q_{i,t}=\frac{S_{i,t}}{\sum_{k\in R_t}S_{k,t}}$$
  其中 $$R_t$$ 表示该周仍未淘汰的所有选手集合。$$q_{i,t}$$ 是评委认为选手 $$i$$ 技术贡献的占比
3. 观众投票建模：未知变量$$v_{i,t}$$
  由于总票数不公开，只需要估算投票比例：$$p_{i,t} = \frac{v_{i,t}}{\sum_{k \in R_t} v_{k,t}}$$
  于是自动满足：$$\sum_{i \in R_t} p_{i,t} = 1$$（每个观众每周的投票次数不确定）
4. 综合得分模型（题目中最后的示例，给出的意思应该是权重为1：1）
  设评委权重为 $$\alpha$$，观众权重为 $$1-\alpha$$。
  综合得分为：$$T_{i,t}=\alpha q_{i,t}+(1-\alpha)p_{i,t}$$
5. 淘汰反推，需要建立约束
  节目公开每周淘汰选手。设该周淘汰者为：$$e(t)$$
  则必须满足：$$T_{e(t),t} \leq T_{i,t}, \quad \forall i \in R_t$$（每一季每一周的选手排名）
  即淘汰者综合得分最低。代入综合得分公式：
  $$\alpha q_{e(t),t} + (1 - \alpha)p_{e(t),t} \leq \alpha q_{i,t} + (1 - \alpha)p_{i,t}$$
6. 仅靠“淘汰最低”约束，存在无穷解（这里应该是约束的重点，GPT给了一种情况）
  1. 平滑人气假设：观众支持通常不会每周剧烈变化，因此设：$$\min \sum_{i,t} (p_{i,t} - p_{i,t-1})^2$$，每一周观众投票的波动尽可能地减少
    - 若一个明星上周票很多，本周票不应突然归零（这里最后求解出来的应该是一个票数区间？对应下面的上下界）加个置信区间是不是就行了
    - 在该种方法下，最后完整的建模为$$\begin{aligned}
\min_{p_{i,t}} \quad & \sum_{i,t} (p_{i,t} - p_{i,t-1})^2 \\\text{s.t.} \quad & \sum_{i \in R_t} p_{i,t} = 1 \\
& p_{i,t} \geq 0 \\
& T_{e(t),t} \leq T_{i,t}, \quad \forall i \\
& T_{i,t} = \alpha q_{i,t} + (1 - \alpha) p_{i,t}
\end{aligned}$$
7. 模型能否正确估算导致淘汰一致的投票？
  1. 淘汰命中率：设总淘汰周数为$$N$$，正确预测淘汰周数为$$M$$，则：$$Accuracy = \frac{M}{N}$$
  2. 淘汰边际（确定性）定义：$$Margin_t = T_{2nd\ lowest,t} - T_{lowest,t}$$
    - Margin 大：淘汰很确定  
    - Margin 小：可能存在争议
8. 每周投票估计确定性是否一样？
  - 在满足约束下，某选手的投票上下界：$$p_{i,t}^{max}, \, p_{i,t}^{min}$$
  - 不确定性：$$U_{i,t} = p_{i,t}^{max} - p_{i,t}^{min}$$
  若 $$U$$ 很大，说明：
  - 投票可能性范围宽  
  - 模型不确定性大

1.2.1.2 两种淘汰规则数学公式
1.2.1.2.1 方法A：百分比法（Percentage Method）
节目Season 3–27采用。1-2和28+需要根据新的来进行猜测
1. Step1：评委占比：$$q_{i,t}=\frac{S_{i,t}}{\sum_{k\in R_t}S_{k,t}}$$
2. Step2：投票占比：$$p_{i,t}=\frac{v_{i,t}}{\sum_{k\in R_t}v_{k,t}}$$
3. Step3：综合百分比得分：$$T^{(perc)}_{i,t}=q_{i,t}+p_{i,t}$$
淘汰者为：$$e_{perc}(t)=\arg\min_{i\in R_t}T^{(perc)}_{i,t}$$
1.2.1.2.2 方法B：排名法（Rank Method）
节目Season 1–2 与 28+采用。其中3-27需要根据Question中得到的$$p_{i,t}$$计算排名
1. Step1：评委排名
  1. 按分数从高到低排序：$$r^J_{i,t}=Rank(S_{i,t})$$
  2. 分数最高者排名1。
2. Step2：观众排名
  1. 按投票从高到低：$$r^V_{i,t}=Rank(v_{i,t})$$
3. Step3：综合排名$$T^{(rank)}{i,t}=r^J{i,t}+r^V_{i,t}$$
  1. 淘汰者为：$$e_{rank}(t)=\arg\max_{i\in R_t}T^{(rank)}_{i,t}$$（排名和最大者最差）

百分比法
$$q_{i,t} = \frac{S_{i,t}}{\sum_{k \in R_t} S_{k,t}}, \quad p_{i,t} = \frac{v_{i,t}}{\sum_{k \in R_t} v_{k,t}}$$
综合：$$T_{i,t}^{perc} = q_{i,t} + p_{i,t}$$
预测“最低者”：$$L_t^{perc} = \arg \min_{i \in R_t} T_{i,t}^{perc}$$
排名法
评委排名（分高排名小）：$$r_{i,t}^J = Rank(S_{i,t})$$
投票排名：$$r_{i,t}^V = Rank(v_{i,t})$$
综合排名和：$$T_{i,t}^{rank} = r_{i,t}^J + r_{i,t}^V$$
预测“最低者”（排名和最大最差）：$$L_t^{rank} = \arg \max_{i \in R_t} T_{i,t}^{rank}$$
定义某一周的结果

\begin{cases} 
1, & L_t^{perc} \neq L_t^{rank} \\
0, & \text{continue}
\end{cases}$$

1.2.1.4 判断“哪种方法更偏向观众？” 
- 题目关键问：是否某种方法更倾向于观众投票？
- 核心思想：如果某方法更常使“高人气低评委”选手走得更远，则该方法更偏向观众。
- 偏向观众指数 Audience Bias Index（ABI）定义选手的“人气-技术差异”：$$Gap_{i,t}=p_{i,t}-q_{i,t}$$（选手评分-评委给分？这里给分的基数不一样吧，哦百分比是一样的，观众的数据都用第一问出来的就行）
  - Gap>0：观众喜欢多于评委认可
  - Gap<0：评委喜欢多于观众支持
- 如果一种方法更偏向观众，它会更倾向于保住Gap大的选手，因此它判为最低/危险的人应当更偏“技术弱、观众也不支持”（Gap更小）。计算两种方法在每周各自判为最低者的Gap：（存在数据缺失即停止）
  - 百分比：$$ABI_{perc}=\frac{1}{N}\sum_t Gap_{e_{perc}(t),t}$$
  - 排名：$$ABI_{rank}=\frac{1}{N}\sum_t Gap_{e_{rank}(t),t}$$
判断：
- 若淘汰者Gap更负 → 方法偏向观众（留下人气选手，因为Gap更小，说明被淘汰的是观众分低的情况）
- 若淘汰者Gap更正 → 方法偏向评委（淘汰人气选手）
例如：$$ABI_{rank}<ABI_{perc} \Rightarrow \text{排名法更偏向观众}$$

1.2.2.2 定义粉丝型选手
定义极端粉丝依赖型选手，满足：  
1. 评委分持续低于中位数： $$z_{i,t} < \text{median}(z_t)$$持续多周  
2. 估计粉丝票显著高于预期：$$\hat{v}_{i,t} > \mathbb{E}[v|z_{i,t}] + 2\sigma_v$$
3. 实际表现远超技术预期（如 Bobby Bones, Bristol Palin）  
识别算法（孤立森林或基于残差的异常检测）：
计算技术-人气偏离度：$$D_i = \frac{1}{T_i} \sum_{t=1}^{T_i} \left( \frac{\hat{v}_{i,t} - \mathbb{E}[v_{i,t} | s_{i,t}]}{\hat{\sigma}_v} \right)$$
选手$$i$$被归类为“粉丝型”，若$$D_i > \tau$$ （阈值，如第 90 百分位数）。
特征分析：
对识别出的选手进行卡方检验或 Fisher 精确检验，分析其行业分布$$g_i$$、年龄组$$a_i$$、地域类型$$r_i$$是否与非粉丝型选手有显著差异。
在强观众支持下，排名法更容易“救活”争议选手，而百分比法更容易让评委低分持续累积。
- 对于评委与观众意见严重分歧的选手，所采用的投票合并方法会显著影响比赛结果。
- 排名法由于其对评委低分的“压缩效应”，更有利于高人气但技术评分较低的选手；
- 而百分比法会持续放大评委评分差异，更倾向于技术导向的结果。
- 引入评委从底二中选择淘汰的机制，能够有效抑制观众投票对结果的极端影响，但同时也削弱了观众参与的决定性作用。
1.2.2.3 检验两种方法：
粉丝型选手的核心特征就是“评委分数低但实际名次高”
你们想证明：排名法（rank-based）由于压缩效应，更容易让这类选手获得更好的最终排名（placement更小）
而百分比法（percent-based）更线性，更容易让评委低分拖累最终排名（placement更大）
定义排名：
如果 d_i > 0 → 排名法下名次更差（数字更大），即百分比法更利于该选手
如果 d_i < 0 → 排名法下名次更好（数字更小），即排名法更救粉丝型选手
如果 d_i ≈ 0 → 两种方法结果相似
d_i = placement_rank_method - placement_percent_method
H₀: 排名法下的placement中位数 ≥ 百分比法下的placement中位数

H₁: 排名法下的placement中位数 < 百分比法下的placement中位数

- 计算所有粉丝型选手的 d_i = placement_rank - placement_percent
- 只看负差值的方向（因为负差值意味着排名法更好）
- 在Python中：wilcoxon(d_i, alternative='less')

import numpy as np
from scipy.stats import wilcoxon

# 示例数据：4个争议选手（你可以替换成你模拟的结果）
# placement越小越好
placement_rank = np.array([2, 5, 3, 1])   # Jerry Rice, Billy Ray, Bristol, Bobby
placement_percent = np.array([5, 8, 7, 4])  # 假设模拟结果

diff = placement_rank - placement_percent
print("差值（rank - percent）：", diff)
# 输出示例： [ -3, -3, -4, -3 ]  全负，说明排名法都更好

# 单侧检验：H1: diff < 0 （排名法 placement 更小，即更好）
stat, p_value = wilcoxon(diff, alternative='less')

print(f"Wilcoxon统计量: {stat}")
print(f"单侧 p 值 (排名法更好): {p_value:.5f}")

if p_value < 0.05:
    print("有显著证据表明：排名法让粉丝型选手的最终名次显著优于百分比法")
else:
    print("没有足够证据表明排名法显著更好（但可以看描述统计）")

1.3.1 评委的分数怎么来的
变量构造（从CSV直接算）
对选手$$i$$在周 $$t$$：
- 评委总分：$$S_{i,t} = \sum_j \text{score}_{i,t,j}$$
- 为了消除每周评委人数不同（3或4），用评委均分： $$J_{i,t} = \frac{S_{i,t}}{\#\text{judges}_t}$$
推荐模型：分层/混合效应线性模型（最合适）
$$J_{i,t} = \beta_0 + \beta_{\text{age}} Age_i + \beta_{\text{ind}}[Industry_i] + \gamma_{\text{pro}}[Pro_i] + \delta_{\text{season}}[Season_i] + \delta_{\text{week}}[Week_t] + \epsilon_{i,t}$$
解释：
- $$\beta_{\text{age}}$$: 年龄对评委得分的影响
- $$\beta_{\text{ind}}$$: 行业对评委得分的影响（运动员/演员/歌手等）
- $$\gamma_{\text{pro}}$$: 职业舞者的“教学/编舞/带飞”能力差异（核心！）
- season/week项控制“不同季、不同周整体给分尺度”变化
